# DecoupleNET
Simple sequential network to decouple a 1D NMR spectrum.

Use the Multi_Peak_Training_FID_Generator script to create a trianing data set that contains multi peak FIDs, and then run the DecoupleNET V2 notebook.

Use the Training_FID_Generator script to create a training dataset that contains single peak FIDs, and then run the DecoupleNET notebook.
